САЙТ BLACK & RED — КАРАОКЕ КЛУБ
================================

Структура проекта:
  index.html        — вся разметка сайта (один лендинг)
  css/style.css     — стили
  js/main.js        — меню, лайтбокс галереи, форма брони
  images/full/      — фотографии зала (1400px)
  images/thumb/     — превью для галереи (680px)
  images/favicon.svg
  Dockerfile        — для деплоя на Amvera


ДЕПЛОЙ НА AMVERA (через git)
================================
1. Создай проект на amvera.ru, тип — Docker (или "из Git").
2. Распакуй архив, перейди в папку проекта в терминале.
3. Выполни:

   git init
   git add .
   git commit -m "feat: добавлен лендинг Black & Red"
   git remote add amvera <ссылка-на-репозиторий-из-личного-кабинета-amvera>
   git push amvera master

4. Amvera соберёт Dockerfile (nginx:alpine + статика) и поднимет сайт.
5. После сборки сайт будет доступен по адресу вида
   https://<имя-проекта>-<юзер>.amvera.io
